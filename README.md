## üöÄ Getting Started (Run Locally)

Want to explore the code or contribute to Sanjeevani? Follow these simple steps to get the app running on your local machine.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (v18 or higher) installed on your computer. 

### Installation Steps

**1. Clone the repository**
Open your terminal and clone the project to your local machine:
```bash
git clone [https://github.com/arooshdatta/Sanjeevani--Emergency-Response-System.git]
```
**2. Navigate to the project directory** 
(Note: If you have separated your files into frontend and backend folders, make sure to cd into the frontend folder!)
```bash
cd Sanjeevani--Emergency-Response-System
# cd frontend (Run this if your React app is inside a frontend folder)
```
**3. Install dependencies** 
Install all required packages using npm (or bun, since this project supports it)
```bash
npm install
```
**4. Start the development server**
Launch the Vite server with hot-module reloading
```bash
npm run dev
```
**5. View the App** 
Open your favorite web browser and navigate to the local server address provided in your terminal (usually http://localhost:5173)
